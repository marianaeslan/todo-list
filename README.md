# 📝 To-Do List  

Um aplicativo simples de lista de tarefas desenvolvido para ajudar na organização do dia a dia.  

## 🚀 Funcionalidades  
- 📌 Adicionar tarefas  
- ✅ Marcar tarefas como concluídas  
- 🗑️ Remover tarefas  
- 💾 Armazenamento local para manter as tarefas salvas  

## 📦 Tecnologias Utilizadas  
- **HTML**  
- **CSS**  
- **JavaScript**
